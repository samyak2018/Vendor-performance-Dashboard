Vendor Performance Analysis Dashboard (Power BI)
Overview

The Vendor Performance Analysis Dashboard is a Power BI project designed to analyze vendor purchasing patterns, product category spending, and procurement efficiency.
This dashboard helps organizations monitor vendor performance, track monthly spending trends, and identify high-performing vendors and product categories.

Features

Monthly spend trend analysis

Total spend by product category

Vendor-wise total spend comparison

Quantity vs Spend analysis

Vendor-wise quantity distribution

Interactive filters for:

Month

Vendor

Product Category
